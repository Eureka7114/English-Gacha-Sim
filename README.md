# English-Gacha-Sim (英语学习激励系统)

这是一个将**离散概率统计模型**与**前端交互**结合的探索项目。

## 项目核心逻辑
* **概率模型**：复现了分段式的抽卡概率增长函数。基础五星概率为 0.6%，在 74 抽后线性增长，直到 80 抽强制收敛（保底）。
* **状态管理**：利用 JavaScript 的 `async/await` 处理复杂的异步抽卡动画与数据状态更新。
* **UI 框架**：采用 Tailwind CSS 进行响应式布局设计。

## 开发者声明 (Important)
本项目在开发阶段尝试了**人机协作模式**：
1.  **逻辑定义**：核心算法逻辑、概率函数设计及 Bug 调试由本人独立完成。
2.  **样式辅助**：部分 UI 样式代码由大语言模型（LLM）辅助生成。
3.  **资源说明**：
    * **关于资源失效**：由于本项目目前采用 GitHub Pages 静态部署，代码中引用的**本地音频与图片路径（D:\桌面\...）**在公网环境下无法加载。
    * **后续计划**：计划将资源迁移至云端 CDN 托管，并引入标准化的路径管理。
